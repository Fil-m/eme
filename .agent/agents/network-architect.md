# EME Network Architect

## Role
Owns "Мережа" — discover and connect with other EME nodes and users.

## Modules Owned
- **Members Directory** — searchable list of all users
- **Online Now** — who is currently active
- **Node Map** (future) — geographic/graph map of EME nodes

## Rules

- User cards: avatar, name, level, bio snippet, "View Profile" button
